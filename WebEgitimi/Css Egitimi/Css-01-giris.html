Css Eğitimi

Css Nedir?

CSS:Cascading style sheet kısaltmasıdır.Ekrandaki html ögelerinin tasarımlarını değiştirmemizi sağlar.

Css Yazma Seçenekleri

  1. Inline : Html elementlerinin içerisine style attribute ü açarak yazılır
  2. Sayfa head alanı içerisinde style etiketleri arasına yazılarak
  3. Harici bir css dosyası oluşturup sayfa head alanı içinde çağırarak

Inline Css Yazımı

Elementlerin açılış taglarına boşluk bırakıp style özelliği içine css kodu yazarak yaparız
Eğer birden fazla css özelliği uygulayacaksak özellikler arasına ; koyarız
En sondakine ; koymaya gerek yoktur.

Head Tagları Arasına Css Yazımı

Sayfa head alanı içerisinde style etiketleri arasına yazılarak sayfadaki elementlere stil uygulanabilir

Harici Css Dosyası Kullanarak Tasarım Uygulama

Harici bir css dosyası oluşturup (style.css gibi ) sayfa head alanı içinde linkl özelliği ile çağırarak tasarımı şekillendirebiliriz

Css Seçiciler

1-Element Seçiciler

Sayfadaki elementleri html taglarına göre (p, h1, div vb) seçerek css uygulayabiliriz
Bu durumda sayfadaki tüm elementler seçilecektir

2-Id Seçici

Sayfadaki elementleri id özelliğine göre seçip stil uygulamamızı sağlar.
Genelde sayfadaki tek bir elementi seçmek için kullanırız
Css dosyası içerisinden # karakteri yazılıp elementin id değeri eklenerek element seçilir.
Örnek: #idSecici{}

3-Class-Sınıf Seçici

Web sayfasında veya uygulama genelinde birden fazla elemente aynı stil ayarlarını uygulamak için class seçicileri kullanırız

Element tagı içerisinde class yazıp css dosyasında yazılan class adını vererek kullanırız.

Css dosyası içerisinde . işaretiyle class oluştururuz.
Örneğin: .sinifSecici{}

Bir elemente class özelliği içerisine aralarına boşluk bırakarak birden fazla class uygulayabiliriz.

Ana div içindeki ilk p

Bir html elementinin içindeki alt elementleri hedefleyerek de stil uygulayabiliriz

Bunun için ana elementi seçip boşluk veya > işaretiyle alt elementleri seçebiliriz. Örneğin: #anaDiv p{}

Ana div içindeki son p

  2. Dış css dosyası veya sayfa içindeki style alanındaki cssler

border double : çift çizgi
